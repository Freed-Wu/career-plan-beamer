career-plan-beamer
==================

A backup of my homework. 职业生涯规划。

Dependent
---------

1.  A LaTex distribution. Such as [texlive].

Install
-------

``` {.zsh}
git clone git@github.com:Freed-Wu/career-plan-beamer
cd career-plan-beamer
latexmk -pvc main.tex
```
